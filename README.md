# Zyrel - Project & Issue Management Platform

Zyrel is a full-stack platform for managing projects, issues, and team collaboration. It provides an intuitive interface to create projects, manage team members, track issues, and streamline workflow using a drag-and-drop Kanban board. Built with **React.js** for the frontend and **Spring Boot** for the backend.  

> ⚠️ **Note:** JWT-based authentication is coming soon. Currently, the app can be run locally for full functionality.
